**FlowForge – Visual Workflow Automation Builder**

FlowForge is a visual automation workflow builder that allows users to create automation pipelines using a drag-and-drop node interface.
Inspired by tools like Node-RED and Zapier, FlowForge enables users to design automation logic by connecting Trigger, Condition, and Action nodes.
The application demonstrates how complex automation workflows can be built visually without writing backend logic.

**Features**
1.Visual drag-and-drop workflow editor
2. Trigger, Condition, and Action nodes
3. Connect nodes to build automation pipelines
4. Interactive node editing
5. Undo / Redo functionality
6. Import and Export workflows
7. Execution console with workflow logs
8. Dark theme interface
9. Workflow persistence using Local Storage

**Example Workflow****
Example automation pipeline:
Temperature Sensor Trigger → Temperature > 30°C → Turn Fan ON
This workflow simulates an IoT automation scenario where an action is triggered based on sensor conditions.
